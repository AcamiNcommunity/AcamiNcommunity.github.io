What is this server about?

This server is about gaming and other fun stuff! We are interested in anime like jjk and others but also make events that Cube hosts,in which you can participate in videos sometimes!

What to expect

Expect the unexpected, there's too much fun to be had here, many channels with people to talk to, many people to talk and make jokes, (do note some people have a more dark style of humour.) many bots to play around with, interactive mods, admins and people running AcamiN and the server, frequent server updates, events and special events that include people in videos!

Who's the host?
